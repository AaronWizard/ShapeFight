# ShapeFight

*AaronWizard, 2019*

An intrepid circle storms the triangle fortress.

Made in Godot 3.1.

## Controls

Arrow keys and WASD keys: Movement and attack
Space: Wait

Mouse: Click empty tile to move there. Click enemy to attack. Click player to wait.
